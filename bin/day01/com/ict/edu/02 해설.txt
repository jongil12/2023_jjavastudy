# 식별자와 예약어

실별자 = 명명법 = 이름 만드는법

클래스 : 첫글자 대문자로 시작한다.

- 하나이상의 단어가 합쳐질때는 단어의 첫글자는 대문자이다.
- 뛰어쓰기 안됨, 특수문자 안됨
- 숫자는 중간이나 뒤에 붙인다.

클래스 안에는 필드와 메서드가 존재한다.

필드 = 상태값 = 데이터

변수와 상수는 필드의 일종입니다.

변수: 프로그램이 끝날 때까지 변할 수 있는 상태값입니다.
상수: 한번 지정하면 프로그램이 끝날때 까지 변하지 않은 상태값

변수는 첫글자 소문자로 시작한다.

하나이상의 단어가 합쳐질 때는 단어의 첫글자들은 대문자 이다.

뛰어쓰기 안됨, 특수문자 안됨, 숫자는 중간이나 뒤에 붙인다.

상수는 모든글자가 대문자이다

하나이상의 단어가 합쳐질 때는 언더스코어 _ 를 사용할 수 있다.

(예시)

JavaLandExam  : 클래스

javaLandExam  : 변수

JAVALANDEXAM  : 상수

JAVA_LAND_EXAM : 상수

메서드 : 기능, 동작 하는 것 메서드 뒤에는 무조건 메서드이름() 가 붙는다.

1. 메서드이름 만드는 법은 변수와 같다. 그래서 소문자이다.
